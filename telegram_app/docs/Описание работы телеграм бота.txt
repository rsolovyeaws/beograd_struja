Описание работы телеграм бота:

1. Бот предлагает пользователю выбрать язык
2. Пользователь выбирает язык (сербский, русский или английский)
3. Бот обьянсяет что делает этот бот
4. Бот предлагает пользователю выбрать действие
   - Добавить адрес
   - Удалить ударес
   - Посмотреть все мои адреса
5. Пользователь выбирает действие
    5а. Пользователь выбрал "Показать все мои адреса"
        5а1. бот присылает пользователю сообщение на выбранном языке, в котором показывает все адреса пользователя из базы данных
        5а2. Бот предлагает пользователю выбрать действие (пункт 4)
    5б. Пользователь выбрал "Удалить адрес"
        5б1. Бот показывает все адреса пользователя и просит выбрать номер адреса, который нужно удалить
            5б1а. Если у пользователя нет адресов в базе данных, то бот присылает пользователю сообщение на выбранном языке, в котором говорит что у пользователя нет адресов
            5б1б. Бот предлагает пользователю выбрать действие (пункт 4)
        5б2. Пользователь вводит(выбирает) номер адреса, который нужно удалить
        5б3. Бот удаляет адрес из базы данных
        5б4. Бот присылает пользователю сообщение на выбранном языке, в котором говорит что адрес удален, и список оставшихся адресов
        5б5. Бот предлагает пользователю выбрать действие (пункт 4)
    5с. Пользователь выбрал "Добавить адрес"
        5а1. Бот присылает пользователю сообщение на выбранном языке, в котором предлагает ввести район Белграда
            5а1а. Польователь вводит район
            5а1б. Бот проверяет есть ли такой район в Белграде
                5а1б1. Если район найден, то бот просит ввести улицу
                    5а1б1а. Пользователь вводит улицу
                    5а1б1б. Бот проверяет есть ли такая улица в районе
                        5а1б1б1. Если улица найдена, то бот просит ввести номер дома
                            5а1б1б1а. Пользователь вводит номер дома
                            5а1б1б1б. Бот проверяет есть ли такой дом на улице
                                5а1б1б1ба. Если дом найден
                                    5а1б1б1ба1. Бот присылает пользователю сообщение на выбранном языке, в котором говорит что все данные введены верно, и возвращает пользователю полный адрес
                                    5а1б1б1ба2. Бот записывает все данные в базу данных
                                    5а1б1б1ба3. Бот прислыает пользоваетлю список всех адресов пользователя и предлагает выбрать действие (пункт 4)
                                5а1б1б1бб. Если дом не найден, бот просит ввести номер дома еще раз (пункт 5а1б1б1)
                        5а1б1б2. Если улица не найдена, бот просит ввести улицу еще раз (пункт 5а1б1)
